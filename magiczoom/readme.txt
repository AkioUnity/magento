INSTALLATION:

NOTE: To manually install Magic Zoom™ please read 'readme_manual.txt'.

IMPORTANT: If you use Magento Compiler, disable it. After installation is complete, go to Magento Compiler and run the compilation process.

1. Unzip the contents of the zip file and upload them to your Magento directory.

2. Go to http://your.site.url/magiczoom/ in your browser and follow the instructions.

3. Go to 'System -> Cache Management' and refresh the cache.

4. Configure the Magic Zoom™ module through the 'Magic Toolbox -> Magic Zoom™ settings' menu in the Magento admin panel.

5. Magic Zoom is ready to use!

6. To upgrade your version of Magic Zoom (which removes the "Please upgrade" text), buy Magic Zoom and overwrite the skin/frontend/[your_interface]/[your_theme]/js/magiczoom.js file file with the new one in your licensed version.
